## System Requirements

<Info>
  - **Node.js**: v18.0.0 or higher - **Operating System**: macOS, Linux, or
  Windows - **Package Manager**: npm, pnpm, or yarn
</Info>

## Install the CLI

<CodeGroup>

```bash npm
npm install -g vulcn
```

```bash pnpm
pnpm add -g vulcn
```

```bash yarn
yarn global add vulcn
```

</CodeGroup>

Verify the installation:

```bash
vulcn --version
# Output: 0.5.0
```

## Install Browsers

Vulcn uses [Playwright](https://playwright.dev) for browser automation. Browsers are installed automatically on first use, but you can install them manually:

```bash
# Install Chromium (default)
vulcn install

# Install specific browsers
vulcn install chromium firefox webkit

# Install all browsers
vulcn install --all
```

## Initialize a Project

Create a `.vulcn.yml` project config:

```bash
vulcn init https://your-target.com
```

This creates a configuration file and `sessions/` directory:

```yaml
# .vulcn.yml
target: https://your-target.com

scan:
  browser: chromium
  headless: true
  timeout: 30000

payloads:
  types:
    - xss

detection:
  xss:
    dialogs: true
    console: true
    severity: high
  reflection:
    enabled: true
  passive: true

crawl:
  depth: 2
  maxPages: 20
  sameOrigin: true

report:
  format: html
```

<Tip>
  You can run `vulcn init --force` to overwrite an existing configuration file.
</Tip>

## Programmatic Usage

For programmatic usage, install the engine package:

<CodeGroup>

```bash npm
npm install @vulcn/engine
```

```bash pnpm
pnpm add @vulcn/engine
```

```bash yarn
yarn add @vulcn/engine
```

</CodeGroup>

Example:

```typescript
import {
  DriverManager,
  PluginManager,
  parseProjectConfig,
} from "@vulcn/engine";
import browserDriver from "@vulcn/driver-browser";

const drivers = new DriverManager();
drivers.register(browserDriver);

// Parse session from YAML
const session = drivers.parseSession(sessionYaml);

// Load plugins from config
const plugins = new PluginManager();
const config = parseProjectConfig({ payloads: { types: ["xss"] } });
await plugins.loadFromConfig(config);

// Execute scan
const result = await drivers.executeScan([session], plugins, {
  headless: true,
});

console.log(`Found ${result.aggregate.findings.length} vulnerabilities`);
```

## Uninstall

To uninstall Vulcn:

```bash
npm uninstall -g vulcn
```

To remove Playwright browsers:

```bash
rm -rf ~/.cache/ms-playwright  # Linux/macOS
```
